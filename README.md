# Solar Energy for Sustainable Development

## About the Project

This project was developed during my first semester as part of the Environmental Chemistry course.
The objective of the project was to create an informative website explaining the importance of solar energy and its applications in different fields such as water treatment, transportation, agriculture, and educational institutions.
The website is designed as a multi-page application where each topic is presented on a separate page connected through a common navigation bar.


## Tech Stack

* HTML
* CSS
* JavaScript


## Features

* Multi-page website
* Navigation bar connecting all pages
* Informative content on solar energy applications
* Hosted using GitHub Pages


## What I Learned

Through this project, I learned:

* Creating and linking multiple HTML pages
* Designing simple and consistent user interfaces
* Structuring content effectively
* Deploying websites using GitHub Pages
